Symfony Boilerplate

A Symfony boilerplate with GraphQL and Nuxt.js

Get Started

Development Environment

Static analysis tools for your code and a Docker Compose stack that will help you building a solid infrastructure in production.

GraphQL API

Thanks to GraphQLite, you can build without hassle a GraphQL API around your PHP use cases.

Nuxt.js

Vue.js with Server-side Rendering (SSR)! Also, no more struggle with forms validation and access control, as everything is done on the API.
